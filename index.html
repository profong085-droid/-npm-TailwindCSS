រៀនប្រើ npm និងដោះស្រាយបញ្ហា TailwindCSS 🚀

ជំនួយដ៏ពេញលេញសម្រាប់អ្នកចាប់ផ្តើម 🎓

npm ជាអ្វី? 🤔

npm (Node Package Manager) គឺជាកម្មវិធីគ្រប់គ្រង packages សម្រាប់ JavaScript។ វាត្រូវបានប្រើប្រាស់យ៉ាងទូលំទូលាយក្នុងការអភិវឌ្ឍ Web Applications ដោយមាន Node.js។

គំរោងថ្មីជាមួយ npm ✨

បង្កើតគំរោងថ្មី

Command Line
mkdir my-project
cd my-project
npm init -y

ដំឡើង Packages 📥

Command Line
npm install <package-name> # ដំឡើង package
npm install --save-dev <package-name> # ដំឡើងសម្រាប់ development
npm install -g <package-name> # ដំឡើងជាសកល

ដំឡើង Node.js និង npm 🔧

npm មកជាមួយ Node.js។ ដូច្នេះអ្នកត្រូវតែដំឡើង Node.js ជាមុនសិន៖

  1. ចូលទៅកាន់ https://nodejs.org
  2. ដំឡើង version LTS (Recommended)
  3. ពិនិត្យការដំឡើងដោយបើក Command Prompt/Terminal
Command Line
node -v
npm -v

ដំឡើង TailwindCSS 🎨

Command Line
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

ការប្រើ Packages 💼

បន្ទាប់ពីដំឡើង package ហើយ អ្នកអាចប្រើវានៅក្នុង code របស់អ្នក៖

JavaScript
// នៅក្នុង file JavaScript របស់អ្នក
const _ = require('lodash');

const numbers = [1, 2, 3, 4, 5];
const sum = _.sum(numbers);
console.log(sum); // Output: 15

Scripts 📜

នៅក្នុង package.json មានផ្នែក scripts ដែលអនុញ្ញាតឱ្យអ្នកដំណើរការ commands ផ្សេងៗ៖

package.json
"scripts": {
"start": "node app.js",
"dev": "nodemon app.js",
"test": "echo \"Error: no test specified\" && exit 1"
}
Command Line
npm run dev
npm start
npm test

បញ្ហាទូទៅជាមួយ TailwindCSS 😫

បញ្ហា: npm error could not determine executable to run

នេះជាបញ្ហាទូទៅនៅពេល npm/npx មិនអាចរកឃើញ binary នៃ tailwindcss។

បញ្ហា: 'tailwindcss' is not recognized

បញ្ហានេះកើតឡើងព្រោះ Windows មិនអាចរកឃើញពាក្យបញ្ជា tailwindcss ក្នុង PATH environment variable របស់អ្នក។

មូលហេតុដែលអាចជា 🔍

  • ការដំឡើង TailwindCSS មិនបានសម្រេច
  • បញ្ហាជាមួយ npm cache
  • ការកំណត់ PATH environment variable មិនត្រឹមត្រូវ
  • កំណែ Node.js/npm មានបញ្ហា

ដំណោះស្រាយបញ្ហា TailwindCSS ✅

១. ដំឡើង TailwindCSS ឡើងវិញ

Command Line
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest

២. សាកល្បងដំណើរការតាមរយៈ node_modules

Command Line
.\node_modules\.bin\tailwindcss.cmd init -p

៣. Clear npm cache

Command Line
npm cache clean --force
npm cache verify

៤. បង្កើតឯកសារកំណត់រចនាសម្ព័ន្ធដោយផ្ទាល់

បង្កើតឯកសារ tailwind.config.js៖

tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
content: ["./src/**/*.{html,js}"],
theme: {
extend: {},
},
plugins: [],
}

បង្កើតឯកសារ postcss.config.js៖

postcss.config.js
module.exports = {
plugins: {
tailwindcss: {},
autoprefixer: {},
},
}
